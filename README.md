# Pedidos Veloz

Projeto desenvolvido para a disciplina de Arquitetura DevOps para Microsserviços.

## Tecnologias

- Node.js
- Docker
- Docker Compose
- PostgreSQL
- GitHub

## Estrutura

- API Gateway
- Serviço de Pedidos
- Serviço de Pagamentos
- Serviço de Estoque

## Execução

docker compose up --build
